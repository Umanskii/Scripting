# Comparison Operators

## String Comparison
- `=` or `==` : Equal
- `!=` : Not equal
- `<` : Less than
- `>` : Greater than

## Integer Comparison
- `-eq` : Equal
- `-ne` : Not equal
- `-lt` : Less than
- `-le` : Less than or equal
- `-gt` : Greater than
- `-ge` : Greater than or equal

## File Comparison
- `-e` : Exists
- `-f` : Is a regular file
- `-d` : Is a directory
- `-r`, `-w`, `-x` : Read, write, execute permissions
